# Run and deploy WAPIMI

This contains everything you need to run the WAPIMI WhatsApp marketing SaaS locally and deploy it to Render.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Create a local `.env` with the Razorpay, app URL, API URL, and allowed origin variables shown in `.env.example`.
3. Run the app:
   `npm run dev`

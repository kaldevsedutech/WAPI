# Run and deploy WAPIMI

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/122921a3-0b2c-4f70-b2c0-167ae600f8b4

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a local `.env` with the Razorpay and app URL variables shown in `.env.example`.
3. Run the app:
   `npm run dev`
